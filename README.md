# goNotes 📝

Легковесное, быстрое и современное self-hosted приложение для заметок с полноценной поддержкой Markdown, вложений и фокусом на персональную продуктивность.

## ✨ Особенности

- **Markdown First**: Продвинутая подсветка синтаксиса, поддержка GFM и быстрое копирование блоков кода в один клик.
- **Цветовое кодирование**: Организуйте свои мысли визуально с помощью адаптивных цветовых схем для заметок.
- **Умное хранение медиа**: Автоматическая генерация превью для изображений (с учетом EXIF), встроенные нативные плееры для видео и аудио.
- **Web Share Target (PWA)**: Полноценный опыт мобильного приложения — "шарьте" ссылки и файлы в goNotes прямо из браузера или галереи смартфона.
- **Мгновенный поиск**: Полнотекстовый поиск, фильтрация по множественным тегам и быстрый доступ к архиву.
- **Self-Hosted & Portable**: Единый бинарный файл, база данных SQLite (WAL mode). Ваши данные принадлежат только вам.
- **Клавиатурный ниндзя**: Ускоряйте работу с помощью горячих клавиш (например, `N` для новой заметки, `E` для редактирования) и полной Tab-навигации.

## 🚀 Технологический стек

- **Backend**: Go 1.25+. Кастомный высокопроизводительный роутер, поддержка Gzip "из коробки".
- **Frontend**: React 19, TypeScript 5, MUI 7. Современный UI с использованием Backdrop Filter Blur и плавных анимаций.
- **Database**: SQLite 3. Оптимизированная схема с индексами для мгновенного поиска по огромным массивам текста.
- **D&D**: Мощная система сортировки заметок и тегов на базе `@dnd-kit`.

## 🛠 Установка и запуск

### Требования
* Go 1.25+
* Node.js 20+ & npm

### 1. Подготовка фронтенда
```bash
cd notes-ui
npm install
npm run release
```
Команда соберет оптимизированный Production-билд и переместит его в директорию для встраивания в Go.

### 2. Сборка и запуск бэкенда
```bash
# Генерация ресурсов и сборка
./scripts/build.resources.sh
go build -o goNotes

# Запуск
./goNotes
```

### 3. Режим разработки
Для работы с горячей перезагрузкой UI:
```bash
# Терминал 1: Бэкенд в режиме DEBUG_UI
./scripts/run.sh dev

# Терминал 2: Фронтенд
cd notes-ui
npm run dev
```

Приложение будет доступно по адресу `http://localhost:80`. Настройки можно изменить в `config.json`, который появится в рабочей директории при первом запуске.

## 📁 Структура проекта

- `/internal` — ядро системы: API эндпоинты, логика обработки медиа и роутинг.
- `/notes-ui` — фронтенд на React: компоненты, хуки и инструменты управления состоянием.
- `/assets` — скомпилированный фронтенд для упаковки в бинарный файл.
- `db.sql` — схема БД с индексами и структурой связей тегов.

## 📱 PWA
Приложение полностью поддерживает манифест PWA. Для установки откройте адрес вашего экземпляра goNotes в браузере и выберите "Добавить на главный экран".
